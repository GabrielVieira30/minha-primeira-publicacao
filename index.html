Publicações Web

O que é uma publicação web, quais os dois tipos de aplicações existentes e suas principais diferenças?

Uma publicação web é a disponibilização de conteúdos na internet, podendo ser estática ou dinâmica. As duas principais aplicações são:

  • Aplicações Estáticas: Conteúdos fixos que não mudam a não ser que sejam editados manualmente.
  • Aplicações Dinâmicas: Conteúdos que mudam de acordo com a interação do usuário ou dados em tempo real.

O que é um domínio e um DNS?

Um domínio é o endereço que identifica um site na internet, enquanto o DNS (Domain Name System) é o sistema que traduz nomes de domínio em endereços IP, permitindo que os navegadores localizem os sites.

Explique o que é um servidor compartilhado e um servidor dedicado, evidenciando suas vantagens e desvantagens.

Um servidor compartilhado é um servidor onde vários sites estão hospedados, enquanto um servidor dedicado é um servidor exclusivo para um único site. Vantagens e desvantagens incluem:

  • Servidor Compartilhado: Vantagem: custo mais baixo. Desvantagem: menos recursos e desempenho.
  • Servidor Dedicado: Vantagem: recursos exclusivos e melhor desempenho. Desvantagem: custo mais elevado.

O que é um servidor web, explique os dois principais servidores. Como fazer o backup de um projeto de forma compactada. Bem como restaurá-la.

Um servidor web é um sistema que armazena e entrega conteúdo web. Os dois principais servidores web são:

  • Apache: Popular e flexível.
  • Nginx: Alto desempenho e eficiente em conexões simultâneas.

Para fazer o backup de um projeto, use zip ou tar no terminal. Para restaurar, descompacte o arquivo.

O que é um servidor de banco de dados, explique como fazer o backup e restaurar esse backup.

Um servidor de banco de dados é um sistema que fornece serviços de gerenciamento de dados. Para fazer o backup, use mysqldump para MySQL. Para restaurar, use o comando apropriado para importar o backup.

O que é um serviço SSH, e como ele permite acessar um servidor web para transferir arquivos de uma máquina local e o servidor.

SSH (Secure Shell) é um protocolo que permite comunicação segura entre computadores. Ele permite acessar um servidor web e transferir arquivos usando scp.

Explique o que é um Linux e cite as principais distribuições usadas como servidor web.

O Linux é um sistema operacional de código aberto baseado no núcleo Linux, que é amplamente utilizado em servidores devido à sua estabilidade, segurança e flexibilidade. Ele permite que os usuários personalizem e modifiquem o sistema de acordo com suas necessidades, sendo uma escolha popular para a hospedagem de sites e aplicações web.

Explique alguns comando básicos do Linux.

Comandos Básicos do Linux pwd: Mostra o caminho completo do diretório atual em que você está. Exemplo: pwd cd: Muda o diretório atual. Você pode especificar um caminho ou usar .. para voltar um diretório. Exemplo: cd /home/usuario ls: Lista todos os arquivos e diretórios no diretório atual. Exemplo: ls ou ls -l para detalhes. mkdir: Cria um novo diretório. Exemplo: mkdir novo_diretorio rm: Remove arquivos ou diretórios. Use -r para remover diretórios recursivamente. Exemplo: rm arquivo.txt ou rm -r diretorio/ cp: Copia arquivos ou diretórios. Exemplo: cp arquivo.txt /destino/ mv: Move ou renomeia arquivos e diretórios. Exemplo: mv arquivo.txt novo_nome.txt cat: Exibe o conteúdo de um arquivo. Exemplo: cat arquivo.txt touch: Cria um arquivo vazio ou atualiza a data de modificação de um arquivo existente. Exemplo: touch novo_arquivo.txt sudo: Executa comandos com privilégios de superusuário, permitindo realizar tarefas administrativas. Exemplo: sudo apt-get update man: Mostra o manual de um comando, explicando suas opções e uso. Exemplo: man ls history: Mostra a lista de comandos que você já executou no terminal. Exemplo: history clear: Limpa a tela do terminal. Exemplo: clear find: Procura arquivos e diretórios com base em critérios específicos. Exemplo: find / -name "arquivo.txt" ps: Mostra os processos em execução no sistema. Exemplo: ps aux
